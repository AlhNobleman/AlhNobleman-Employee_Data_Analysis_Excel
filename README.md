# Employee Data Analysis & Dashboard Project

Cleaned messy HR data → built an interactive Excel dashboard that reveals salary trends, performance insights, and hiring patterns across departments.

**Key Achievements**
- Transformed raw, inconsistent employee records (duplicates, missing values, formatting issues) into clean, reliable data
- Used advanced Excel features: PivotTables, slicers, calculated fields, conditional formatting
- Created dynamic visualizations showing average salaries by department (e.g., IT leads at ~$88k), performance vs. tenure, and yearly hiring trends
- Demonstrated end-to-end skills: data cleaning, transformation, analysis & storytelling

**Skills Highlighted**  
Excel (PivotTables/Charts, formulas, data validation) • Data Cleaning • Dashboard Design • Business Insights from HR Data

## How to View

1. Open `Employee_data.xlsx`
2. Go to **Dashboard** sheet
3. Use slicers to filter

**If Download gives you an HTML file instead** (rare GitHub glitch):
- Right-click the **Raw** button → "Save link as..." → name it `Employee_data.xlsx`
- Or download the whole repository as ZIP:
  1. Click green **Code** button
  2. Choose **Download ZIP**
  3. Extract → open `Employee_data.xlsx`

Go to **Dashboard** sheet and use the slicers to explore!
